Hello there...

Welcome to my todo manager. This app allows you to create, edit, update and delete todos. You have four panels into which you can drag your todos depending on which stage they're on. The drag feature is available to desktop only but you can use the little tabs at the bottom of each todo to navigate the todo from one panel to another on mobile and desktop.

If this is your first time using this please clear your local storage to make sure that you have access to all of the new features. Besides that if you have any suggestion for how I could improve this go to home page and drop me a line or two.
